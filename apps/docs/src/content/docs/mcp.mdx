## Two install paths

1. **Hosted (default)** — paste `https://mcp.nlqdb.com/mcp` into your host's MCP-connector config; OAuth opens in your browser on first tool call.
2. **Local stdio** — `nlq mcp install` auto-detects installed hosts and writes scoped `sk_mcp_<host>_<device>_*` keys into each config.
