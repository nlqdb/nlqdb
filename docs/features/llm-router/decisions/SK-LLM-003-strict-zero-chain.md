# SK-LLM-003 — Day-1 strict-$0 chain: Gemini Flash → Groq → Workers-AI → OpenRouter free

this body is verbatim, only the location moved.

- **Decision:** Until startup credits land, the `plan` tier chain is `[gemini_flash_free, groq_gptoss120b_free, openrouter_free]` (with `workers_ai` as a non-US backup). The `route` tier uses `groq_gptoss20b_free` with `workers_ai` as the geo backup. Embeddings use Workers AI bge-base-en-v1.5. The chain is configured via env var, not code. **Groq model slugs (2026-08-16 migration):** Groq decommissioned `llama-3.1-8b-instant` and `llama-3.3-70b-versatile` on 2026-08-16; this chain follows Groq's own recommended replacements — `openai/gpt-oss-20b` for the cheap (route / engine_classify) tier and `openai/gpt-oss-120b` for the planner (plan / summarize / schema_infer) tier. The slugs live once in `packages/llm/src/providers/groq.ts`.
- **Core value:** Free, Bullet-proof, Honest latency
- **Why:** Every provider in the chain has a no-card free tier (per `docs/architecture.md §7.1`): Gemini 500 RPD plan / 250k TPM, Groq 1,000 RPD per model on `gpt-oss-20b` / `gpt-oss-120b`, Workers AI 10k Neurons/day, OpenRouter 50 RPD anon / 1,000 RPD after a one-time $10 deposit. Stacked, this gives ~500 plan generations + ~10k routings per day (Groq cheap tier 1,000 RPD, then Workers-AI + OpenRouter failover) — comfortably above Phase 1's exit criteria after the plan cache (60–80% hit rate). Groq's deprecation dropped the cheap-tier per-model cap from 14,400 RPD (old `llama-3.1-8b-instant`) to 1,000 RPD, so the failover legs now carry more of the classify volume; the chain order is unchanged. Card-free is the activation guarantee in `GLOBAL-013`.
- **Consequence in code:** Day-1 deploy reads `LLM_CHAIN_PLAN`, `LLM_CHAIN_ROUTE`, `LLM_CHAIN_SUMMARIZE` env vars; defaults are the strict-$0 chain. All four free providers are implemented in `packages/llm/src/providers/`; rotating the chain is a redeploy, not a code change.
- **Alternatives rejected:** Single free provider — one outage kills the product. Wait for credits — punts launch by weeks; `docs/architecture.md §0` says we ship without spending money.
