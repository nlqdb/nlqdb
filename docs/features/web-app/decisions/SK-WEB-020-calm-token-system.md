# SK-WEB-020 — Calm token system (supersedes SK-WEB-015's quiet-brutalism tokens)

- **Decision:** `apps/web/src/styles/global.css` is re-based from the SK-WEB-015 "quiet-brutalism" ladder to a **calm** system, applied across every page. **Neutrals** are near-black, very-low-chroma with a whisper of warmth (`--ground #0b0c0b`, `--bg-2 #0f110f`, `--raised #141613`, `--raised-2 #1b1e1a`, `--rule #232622`, `--rule-2 #30342f`, `--muted #8b908a`, `--body #c6c9c3`, `--ink #f2f4f1`) on Radix-style 12-step roles. **One accent** — jade `--accent #3ecf8e` (+`--accent-2`, `--accent-ink #04210f`, `--accent-soft`, `--accent-line`) — keeps the brand's green DNA without the acid lime, used at ~10% (60-30-10): one promoted CTA per band, links-on-hover, small glyphs. **Faces:** `--sans` is a system stack (no webfont fetch) used for body *and* display (`--display` aliases `--sans`); `--mono` stays for code / identifiers / data / traces. The Source Serif 4 `@font-face` is dropped. **Radius** scale `--r-xs 6 / --r-sm 8 / --r-md 12 / --r-lg 16 / --r-xl 24 / --r-pill` (`--radius` = 12px) replaces the 0px corners. **Shadows** are two-layer key+ambient (`--shadow-1/2/3`, `--ring`) replacing hard offset shadows. **Motion** tokens `--ease` (expo-out `cubic-bezier(0.19,1,0.22,1)`), `--ease-std`, `--dur-1/2/3`; entrance reveals are progressive-enhancement + `prefers-reduced-motion`-gated. Legacy aliases (`--shadow-hard`, `--border-strong`, `--bg-elev`, …) are remapped to the calm ladder so any un-swept `<style>` block renders calm automatically.
- **Core value:** Effortless UX, Honest latency, Bullet-proof
- **Why:** The brutalist look (acid lime, 0 radius, hard offset shadows, mono-as-display) is the visual opposite of the product's job — to feel *relaxed, trustworthy, secure, and fast*. Modern dev-tool research is convergent: calm = low-saturation near-black neutrals + one restrained accent + generous whitespace + soft layered shadows + ~12px radius (Vercel Geist, Supabase `#3ecf8e`, Linear LCH-neutral, Radix scale), and visual craft is itself a trust signal (NN/G; Stanford — 46% judge credibility on visual design; Lindgaard — 50 ms first impression). System fonts are both the modern default (Linear) and a speed win (no webfont fetch). SK-WEB-015's diagnosis (one accent moment, mono demoted, one token source) is right and is *retained in spirit*; only the visual vocabulary changes.
- **Consequence in code:** Tokens + shared primitives (`.cta`, `.btn`, `.topnav`, `.foot`, hero input, `.createform`, `.demo`, `.nlq-*`, error/boot panels) live in `global.css` and carry the calm vocabulary; per-component `<style>` blocks use only these tokens (no new tokens, no hard-coded brutalist values). The home (`/`) layout lives in `apps/web/src/styles/home2.css` scoped under `.home2`. Product surfaces (`styles/chat.css`, `styles/keys.css`) and every marketing page are swept to soft borders (1px `--rule`/`--rule-2`), radius, layered shadows, `--ring` focus, and `--accent-soft` tints. PRs that reintroduce acid lime, 0 radius, hard offset shadows, thick (2–3px) borders, or mono-as-display are rejected.
- **Alternatives rejected:**
  - **Light reset to near-white** — near-black is the brand's most recognisable property in screenshot/share contexts; calm is achievable on dark, and a light mode can be added later without re-basing the ladder.
  - **Keep brutalism, tune rhythm only** — the acid lime + hard shadows are the trust-and-calm problem at root; rhythm tuning leaves it in place (same conclusion SK-WEB-015 reached for mono).
  - **A second/brighter accent for "energy"** — loud accents only land surrounded by quiet; one jade at ~10% restores signal value (carried from SK-WEB-015).
  - **Bespoke/variable webfont (Geist-style)** — table-stakes for that tier, but a webfont fetch costs first paint; a system stack is calm, modern, and free. Revisit only if brand needs a custom face.
